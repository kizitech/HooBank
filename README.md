- Clean banking landing page with a strong hero section
- Responsive layout across desktop and mobile
- Feature blocks, billing section, testimonials, and CTA areas
- Smooth, polished UI focused on trust, clarity, and conversion

## Getting Started

### 1. Clone the repo
```bash
git clone https://github.com/kizitech/HooBank.git
cd HooBank
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the app
```bash
npm run dev
```

The app runs locally at `http://localhost:5173`.
